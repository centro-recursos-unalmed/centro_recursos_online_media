Instructions for installing Global eWorkBook - Elementary


Windows

The installation process will begin automatically on most Windows
computers. If you have autorun turned off, Please select the DVD-ROM
drive and double click install from the Install_Win folder. Follow the
onscreen instructions.

Once the installation is complete, an icon will be created on the
desktop. To run the application click on the icon.


Macintosh 

Please select the DVD-ROM drive and double click Install from the
Install_Mac folder. Follow the on-screen instructions.

Once installed, you may wish to drag the application icon from the
applications folder to your dock for easy access. Alternatively, double
click the application from the applications folder to launch.


To view Adobe Acrobat PDF files, Adobe Reader is required.
If Adobe Reader is not already installed, Please install it from
Adobe Reader folder on the DVD
